1. Unzip the folder into your localhost roots directory ('www' for wampp and 'htdocs' for xampp).

2.Create database 'shop' in your phpmyadmin.

3.Click on the 'shop' database you created and click import.

4.Browse into your root directory and select shopDB (zipped folder)

5.Click 'Go' beneath the page.

6.Enter The Url 'localhost/shop' to access teller panel

7.Enter The Url 'localhost/shop/admin' to access admin panel
